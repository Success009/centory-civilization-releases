# Centory Civilization — Official Releases

This repository contains the official, platform-optimized compiled release binaries for the **Centory Civilization Mod**.

These builds are automatically compiled and separated to ensure optimal performance and minimal download sizes:
- **Linux Optimized Build**: Contains only `libbridge-linux-amd64.so`
- **Windows Optimized Build**: Contains only `bridge-windows-amd64.dll`
- **macOS Optimized Build**: Contains only `libbridge-darwin-amd64.dylib`

The mod features a fully functional network gateway layer (automatic tunneling bridge) and a robust self-healing auto-updater system.
